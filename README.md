# Credit Scoring Model (Machine Learning)

## 📌 Project Overview
This project predicts whether a customer is creditworthy or not based on:
- Income
- Debt
- Payment History

## 🧠 Technologies Used
- Python
- Pandas
- Scikit-learn

## ⚙️ Model Used
- Random Forest Classifier

## 📊 Results
The model was trained and tested, achieving good accuracy.

## 🚀 Features
- Data preprocessing
- Model training
- Accuracy evaluation
- New customer prediction

Open the notebook in Google Colab and run all cells.
